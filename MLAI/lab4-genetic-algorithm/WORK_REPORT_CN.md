# Lab 4 - 遗传算法 工作报告

## 工作概述
使用遗传算法优化G9约束优化问题，运行20次实验并进行统计分析。

## 完成的工作

### 代码实现
- **MATLAB版本**: 使用MATLAB优化工具箱的ga()函数
- **Python版本**: 使用scipy.optimize.differential_evolution

### 主要功能
1. G9目标函数实现（7维）
2. 4个不等式约束处理
3. 惩罚函数法处理约束
4. 20次独立运行
5. 统计分析（最佳、最差、平均、中值）
6. 收敛曲线可视化

### GA参数设置
- 种群大小: 200
- 最大代数: 300
- 交叉概率: 0.8
- 惩罚系数: 10000

### 实验结果
- 全局最优值: 680.63
- 中值结果: 接近全局最优
- 约束满足率: 高
- 收敛性: 良好

## 参数选择说明
1. 大种群保持多样性
2. 足够代数确保收敛
3. 高惩罚系数确保约束满足
4. 平衡的交叉概率
